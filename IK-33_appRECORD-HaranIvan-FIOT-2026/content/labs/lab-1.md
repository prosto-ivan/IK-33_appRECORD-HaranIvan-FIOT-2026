**Тема:** Створення адаптивного інтерфейсу з використанням CSS Grid та системи ролей.
**Мета:** Розробити прототип освітньої платформи з підтримкою темної теми та фільтрації контенту.

## 📸 Галерея інтерфейсу

### 1. Доступ Гостя (Світла тема)
Основний каталог доступний для перегляду всім. У цьому режимі історія відвідувань не зберігається локально.
![Вид гостя](/assets/labs/lab-1/guest.png)

### 2. Доступ Користувача (Темна тема)
Реалізовано Dark Mode. Користувач має доступ до вкладки «Моє навчання» та персоналізованого списку останніх відвіданих ресурсів.
![Вид користувача темна тема](/assets/labs/lab-1/black_user.png)

### 3. Панель Адміністратора
Адмін має розширені права, що включають можливість модерації карток та доступ до системних налаштувань.
![Вид адміна](/assets/labs/lab-1/admin.png)

### 4. Адаптивний дизайн (Mobile)
За допомогою медіа-запитів (`@media`) інтерфейс перелаштовується в одну колонку для зручного користування зі смартфонів.
![Мобільна версія](/assets/labs/lab-1/mobile.png)

## 🛠 Технічні деталі

### Робота з сіткою (CSS Grid)
Використано для розділення основного контенту та сайдбару:
```css
.layout-wrapper {
    display: grid;
    grid-template-columns: 3fr 1fr;
    gap: 30px;
    align-items: start;
} 
```
## 🛠 Логіка перемикання ролей (JavaScript)

Для реалізації контролю доступу без використання важких фреймворків було застосовано підхід маніпуляції класами кореневого елемента `body`. Це дозволяє синхронізувати стан JS із CSS-правилами.

```javascript
// Основна функція входу та зміни прав доступу
function login(role) {
    // Оновлення глобального стану
    state.currentUserRole = role;

    // Перемикання екранів (Авторизація -> Додаток)
    document.getElementById('auth-page').style.display = 'none';
    document.getElementById('main-app').style.display = 'block';

    // Динамічне керування класами для CSS-селекторів
    document.body.className = `${state.isDarkTheme ? 'dark-theme' : 'light-theme'} role-${role}`;

    // Перемальовування контенту згідно з новою роллю
    renderCards();
}
